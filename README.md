# Junk-Droids-Spider-Robot
Repository for Junk Droids Spider robot. Will include readme and documentation in future explaining how SW works.

## Important notes
- specify upload_port from computer you're using
 - Linux: upload_port = /dev/ttySn
 - Sn is the comport associated with your WIndows dev.
 - S3 -> COM3
 - S5 -> COM5
